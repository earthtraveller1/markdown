# Markdown

A basic program that can parse a markdown file and produce an HTML document. It is currently in development and not recommended for production use, though feel free to try it out if you would like.

## Installing

Currently, this program is not available for installation, though that may change in the future. You can still build this from source by installing the Rust toolchains, cloning this repository, and running `cargo run` to run the program.

There are no features just yet.
